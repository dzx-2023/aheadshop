## 3. 分层架构设计

### 3.1 分层说明

```
Controller 层
    │  接收请求、参数校验、调用 Service、返回 Result
    │  注解: @RestController, @Validated
    ▼
Service 层
    │  业务逻辑、事务管理、调用 Mapper
    │  注解: @Service, @Transactional
    ▼
Mapper 层
    │  数据访问，继承 BaseMapper<T>
    │  复杂 SQL 用 XML，简单 CRUD 用 MyBatis-Plus 内置方法
    ▼
Entity 层
    │  数据库映射实体，继承 BaseEntity
    │  注解: @TableName, @TableField
```

### 3.3 Service 示例

```java
@Service
@RequiredArgsConstructor
public class SpuServiceImpl extends ServiceImpl<SpuMapper, Spu> implements SpuService {

    private final SkuMapper skuMapper;
    private final SpuMapper spuMapper;

    @Override
    @Transactional(rollbackFor = Exception.class)
    public void saveSpu(SpuSaveDTO dto) {
        // 1. 保存 SPU
        Spu spu = BeanUtil.copyProperties(dto, Spu.class);
        spuMapper.insert(spu);

        // 2. 保存 SKU 列表
        dto.getSkus().forEach(skuDTO -> {
            Sku sku = BeanUtil.copyProperties(skuDTO, Sku.class);
            sku.setSpuId(spu.getId());
            skuMapper.insert(sku);
        });
    }

    @Override
    public IPage<SpuVO> queryPage(SpuQueryDTO query) {
        return spuMapper.selectSpuPage(
            new Page<>(query.getPageNum(), query.getPageSize()),
            query
        );
    }
}
```

## 5. 安全设计

### 5.1 JWT 认证流程

```
用户登录
    │
    ▼
验证用户名密码
    │
    ▼
生成 JWT Token（含 userId、角色）
    │
    ├── AccessToken  (有效期 2h，存 Header)
    └── RefreshToken (有效期 7d，存 Redis)
    │
    ▼
前端请求携带 Authorization: Bearer {token}
    │
    ▼
Gateway 全局过滤器校验 Token
    │
    ├── 白名单放行 (/api/user/login, /api/user/register, /api/product/**)
    └── 校验失败返回 401
```

### 5.2 接口鉴权

```java
// Gateway 全局鉴权过滤器
@Component
public class AuthGlobalFilter implements GlobalFilter, Ordered {

    private static final List<String> WHITE_LIST = Arrays.asList(
        "/api/user/login",
        "/api/user/register",
        "/api/product/"
    );

    @Override
    public Mono<Void> filter(ServerWebExchange exchange, GatewayFilterChain chain) {
        String path = exchange.getRequest().getURI().getPath();

        // 白名单放行
        if (isWhiteListed(path)) {
            return chain.filter(exchange);
        }

        // 获取 Token
        String token = exchange.getRequest().getHeaders().getFirst("Authorization");
        if (StringUtils.isBlank(token)) {
            return unauthorized(exchange);
        }

        // 校验 JWT
        try {
            Claims claims = JwtUtil.parseToken(token.replace("Bearer ", ""));
            // 将用户信息放入请求头传递给下游服务
            ServerHttpRequest request = exchange.getRequest().mutate()
                .header("X-User-Id", claims.getSubject())
                .header("X-User-Role", claims.get("role", String.class))
                .build();
            return chain.filter(exchange.mutate().request(request).build());
        } catch (Exception e) {
            return unauthorized(exchange);
        }
    }
}
```

### 6.2 缓存工具类

```java
@Component
@RequiredArgsConstructor
public class CacheService {

    private final StringRedisTemplate redisTemplate;

    public <T> void set(String key, T value, long timeout, TimeUnit unit) {
        redisTemplate.opsForValue().set(key, JSONUtil.toJsonStr(value), timeout, unit);
    }

    public <T> T get(String key, Class<T> clazz) {
        String json = redisTemplate.opsForValue().get(key);
        return json == null ? null : JSONUtil.toBean(json, clazz);
    }

    public Boolean delete(String key) {
        return redisTemplate.delete(key);
    }

    public Long increment(String key) {
        return redisTemplate.opsForValue().increment(key);
    }
}
```

### 6.3 缓存穿透/击穿/雪崩防护

```java
// 防缓存穿透：布隆过滤器 + 空值缓存
// 防缓存击穿：分布式锁（Redisson）
// 防缓存雪崩：随机过期时间

public SpuDetailVO getSpuDetail(Long spuId) {
    String key = "product:detail:" + spuId;

    // 1. 查缓存
    SpuDetailVO vo = cacheService.get(key, SpuDetailVO.class);
    if (vo != null) {
        return vo;
    }

    // 2. 缓存未命中，加分布式锁
    String lockKey = "lock:product:" + spuId;
    RLock lock = redissonClient.getLock(lockKey);
    try {
        lock.lock();
        // 双重检查
        vo = cacheService.get(key, SpuDetailVO.class);
        if (vo != null) return vo;

        // 3. 查数据库
        vo = spuMapper.selectDetail(spuId);
        if (vo == null) {
            // 空值缓存防穿透
            cacheService.set(key, new SpuDetailVO(), 1, TimeUnit.MINUTES);
        } else {
            // 随机过期时间防雪崩
            long ttl = 10 + RandomUtil.randomInt(5);
            cacheService.set(key, vo, ttl, TimeUnit.MINUTES);
        }
        return vo;
    } finally {
        lock.unlock();
    }
}
```

### 7.2 库存扣减方案

```java
// 乐观锁扣库存（适合并发不高的场景）
@Update("UPDATE t_sku SET stock = stock - #{quantity}, " +
        "locked_stock = locked_stock + #{quantity} " +
        "WHERE id = #{skuId} AND stock >= #{quantity}")
int lockStock(@Param("skuId") Long skuId, @Param("quantity") Integer quantity);

// Redis 预扣库存（适合高并发场景）
public boolean deductStock(Long skuId, int quantity) {
    String key = "stock:" + skuId;
    Long result = redisTemplate.opsForValue().decrement(key, quantity);
    if (result != null && result >= 0) {
        return true;
    }
    // 库存不足，回补
    redisTemplate.opsForValue().increment(key, quantity);
    return false;
}
```

```java
@PostMapping("/callback/alipay")
public String alipayCallback(HttpServletRequest request) {
    Map<String, String> params = AlipayUtils.parseParams(request);

    // 1. 验签
    if (!AlipaySignature.rsaCheckV1(params, alipayPublicKey, "UTF-8", "RSA2")) {
        return "fail";
    }

    // 2. 幂等检查（防重复回调）
    String payNo = params.get("out_trade_no");
    PayRecord record = payRecordMapper.selectByPayNo(payNo);
    if (record.getStatus() == 1) {
        return "success"; // 已处理
    }

    // 3. 更新支付状态
    record.setStatus(1);
    record.setTradeNo(params.get("trade_no"));
    record.setPayTime(LocalDateTime.now());
    payRecordMapper.updateById(record);

    // 4. 发送 MQ 通知订单服务更新状态
    rabbitTemplate.convertAndSend("pay-exchange", "pay.success", new PaySuccessMsg(payNo));

    return "success";
}
```

### 9.2 ES 查询构建

```java
@Service
public class SearchServiceImpl implements SearchService {

    private final ElasticsearchRestTemplate esTemplate;

    @Override
    public SearchPageResult<ProductDoc> search(SearchParam param) {
        NativeSearchQueryBuilder builder = new NativeSearchQueryBuilder();

        // 关键词查询
        if (StrUtil.isNotBlank(param.getKeyword())) {
            builder.withQuery(QueryBuilders.multiMatchQuery(param.getKeyword(),
                "title", "categoryName", "brandName"));
        }

        // 分类过滤
        if (param.getCategoryId() != null) {
            builder.withFilter(QueryBuilders.termQuery("categoryId", param.getCategoryId()));
        }

        // 价格区间
        if (param.getMinPrice() != null || param.getMaxPrice() != null) {
            RangeQueryBuilder range = QueryBuilders.rangeQuery("price");
            if (param.getMinPrice() != null) range.gte(param.getMinPrice());
            if (param.getMaxPrice() != null) range.lte(param.getMaxPrice());
            builder.withFilter(range);
        }

        // 排序
        switch (param.getSort()) {
            case "price_asc":  builder.withSort(SortBuilders.fieldSort("price").order(SortOrder.ASC)); break;
            case "price_desc": builder.withSort(SortBuilders.fieldSort("price").order(SortOrder.DESC)); break;
            case "sales":      builder.withSort(SortBuilders.fieldSort("sales").order(SortOrder.DESC)); break;
            default:           builder.withSort(SortBuilders.scoreSort().order(SortOrder.DESC));
        }

        // 分页
        builder.withPageable(PageRequest.of(param.getPageNum() - 1, param.getPageSize()));

        return esTemplate.search(builder.build(), ProductDoc.class);
    }
}
```

## 11. 开发规范

### 11.1 代码规范
- 遵循《阿里巴巴 Java 开发手册》
- 使用 Lombok 简化 POJO
- 统一使用 Result 包装响应
- 参数校验使用 @Validated + JSR 303 注解
- 复杂业务逻辑拆分 Service，Controller 保持轻薄

### 11.2 分支规范
- `main` — 生产分支
- `dev` — 开发分支
- `feature/xxx` — 功能分支
- `hotfix/xxx` — 紧急修复分支

### 11.3 提交规范
```
feat: 新功能
fix: 修复
docs: 文档
style: 格式
refactor: 重构
test: 测试
chore: 构建/工具
```

### 11.4 接口文档
- 使用 Knife4j (Swagger 增强) 自动生成
- 所有接口必须添加 @Operation 注解说明
- 本地访问：`http://localhost:8081/doc.html`
