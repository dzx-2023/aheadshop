## 3. 各模块详细设计

### 3.1 公共模块（aheadshop-common）

不独立部署，被其他模块依赖。

```
aheadshop-common
├── common-core          # 通用工具类、统一响应体 Result<T>
├── common-redis         # Redis 配置与工具
├── common-mybatis       # MyBatis-Plus 配置、分页插件、自动填充
├── common-swagger       # Knife4j/SpringDoc 配置
└── common-exception     # 全局异常处理
```

**核心类设计：**

```java
// 统一响应体
public class Result<T> {
    private int code;       // 状态码
    private String msg;     // 提示信息
    private T data;         // 业务数据
}

// 全局异常处理
@RestControllerAdvice
public class GlobalExceptionHandler {
    // BusinessException / MethodArgumentNotValidException / ...
}
```

---

### 3.2 网关服务（aheadshop-gateway）

**职责：**
- 统一路由转发
- JWT Token 校验（白名单放行登录/注册接口）
- 请求限流（Sentinel / Redis + Lua）
- 跨域处理
- 请求日志记录

**路由配置示例：**

```yaml
spring:
  cloud:
    gateway:
      routes:
        - id: user-service
          uri: lb://aheadshop-user
          predicates:
            - Path=/api/user/**
        - id: product-service
          uri: lb://aheadshop-product
          predicates:
            - Path=/api/product/**
        - id: order-service
          uri: lb://aheadshop-order
          predicates:
            - Path=/api/order/**
```

### 3.3 用户服务（aheadshop-user）

**功能模块：**

```
用户服务
├── 注册模块
│   ├── 手机号注册（短信验证码）
│   └── 邮箱注册
├── 登录模块
│   ├── 账号密码登录
│   └── Token 刷新
├── 用户信息模块
│   ├── 个人信息查询/修改
│   └── 头像上传（OSS）
├── 收货地址模块
│   ├── CRUD
│   └── 默认地址设置
└── 权限模块
    ├── RBAC 角色管理
    └── 菜单权限管理
```

**核心接口：**

| 接口 | 方法 | 路径 | 说明 |
|------|------|------|------|
| 注册 | POST | /api/user/register | 用户注册 |
| 登录 | POST | /api/user/login | 返回 JWT Token |
| 获取用户信息 | GET | /api/user/info | 需鉴权 |
| 修改用户信息 | PUT | /api/user/info | 需鉴权 |
| 收货地址列表 | GET | /api/user/address/list | 需鉴权 |

### 3.4 商品服务（aheadshop-product）

**功能模块：**

```
商品服务
├── 分类管理
│   ├── 三级分类树
│   └── 分类关联品牌
├── 品牌管理
│   └── CRUD + Logo 上传
├── SPU 管理
│   ├── 基本信息（标题、副标题、图片列表）

│   ├── 规格属性组合

```

```
├── SKU 2: 颜色=黑色, 存储=256G, 价格=6999, 库存=80
├── SKU 3: 颜色=白色, 存储=128G, 价格=5999, 库存=60
└── SKU 4: 颜色=白色, 存储=256G, 价格=6999, 库存=50
```

### 3.5 购物车服务（aheadshop-cart）

**存储方案：**
- 未登录：Redis Hash，key = `cart:anon:{uuid}`
- 已登录：Redis Hash，key = `cart:user:{userId}`
- 登录后合并离线购物车

**核心接口：**

| 接口 | 方法 | 路径 | 说明 |
|------|------|------|------|
| 添加商品 | POST | /api/cart/add | skuId + 数量 |
| 查询购物车 | GET | /api/cart/list | 含 SKU 详情 |
| 修改数量 | PUT | /api/cart/update | skuId + 新数量 |
| 删除商品 | DELETE | /api/cart/delete/{skuId} | 移除 |
| 选中/取消 | PUT | /api/cart/check/{skuId} | 勾选结算 |

### 3.6 订单服务（aheadshop-order）

**订单状态机：**

```
                  30min超时
  待支付 ──────────────────→ 已取消
    │
    │ 支付成功
    ▼
  已支付 ──→ 商家发货 ──→ 已发货 ──→ 用户收货 ──→ 已完成
    │                                          │
    │ 申请退款                                  │ 申请售后
    ▼                                          ▼
  退款中 ──→ 已退款                        售后处理中
```

**核心接口：**

| 接口 | 方法 | 路径 | 说明 |
|------|------|------|------|
| 创建订单 | POST | /api/order/create | 锁库存、生成订单 |
| 订单列表 | GET | /api/order/list | 分页、按状态筛选 |
| 订单详情 | GET | /api/order/detail/{orderNo} | — |
| 取消订单 | PUT | /api/order/cancel/{orderNo} | 释放库存 |
| 确认收货 | PUT | /api/order/confirm/{orderNo} | — |

**分布式事务：** 订单创建涉及「扣减库存 + 清理购物车 + 生成订单」，采用 RabbitMQ 事务消息保证最终一致性。

### 3.7 支付服务（aheadshop-pay）

**功能模块：**

```
支付服务
├── 发起支付

│   └── 更新订单状态 + 发送 MQ 通知
├── 支付查询
└── 退款
    ├── 退款申请
    └── 退款回调
```

### 3.8 搜索服务（aheadshop-search）

**ES 索引设计：**

```json
{
  "mappings": {
    "properties": {
      "skuId":      { "type": "long" },
      "spuId":      { "type": "long" },
      "title":      { "type": "text", "analyzer": "ik_max_word" },
      "category":   { "type": "keyword" },
      "brand":      { "type": "keyword" },
      "price":      { "type": "double" },
      "sales":      { "type": "long" },
      "score":      { "type": "double" },
      "image":      { "type": "keyword" },
      "attrs":      { "type": "nested" }
    }
  }
}
```

**数据同步：** 商品上架时通过 RabbitMQ 将数据同步到 ES。

---

### 3.9 后台管理服务（aheadshop-admin）

聚合调用各微服务接口，提供后台管理能力。本身不直接操作数据库，通过 Feign 调用各业务服务。

## 4. 模块依赖关系

```
aheadshop-gateway
    └── common-core, common-redis

aheadshop-user
    └── common-core, common-mybatis, common-redis, common-swagger, common-exception

aheadshop-product
    └── common-core, common-mybatis, common-redis, common-swagger, common-exception

aheadshop-cart
    └── common-core, common-redis

aheadshop-order
    └── common-core, common-mybatis, common-redis, common-swagger, common-exception
    └── Feign → product-service, cart-service, user-service

aheadshop-pay
    └── common-core, common-mybatis, common-redis

aheadshop-search
    └── common-core

aheadshop-admin
    └── common-core
    └── Feign → 所有业务服务
```

## 5. 技术选型

| 层次 | 技术                       | 说明 |
|------|--------------------------|------|
| 前端框架 | Vue 3 + TypeScript       | 组合式 API |
| UI 组件库 | Element Plus             | 后台管理 |
| 前端构建 | Vite                     | 快速热更新 |
| 状态管理 | Pinia                    | Vue 3 官方推荐 |
| HTTP 客户端 | Axios                    | 请求拦截、Token 注入 |
| 后端框架 | Spring Boot 3.x          | 主框架 |
| 微服务 | Spring Cloud 2022.x      | 服务治理 |
| 网关 | Spring Cloud Gateway     | 响应式网关 |
| ORM | MyBatis-Plus 3.5+        | 增强 CRUD |
| 数据库 | MySQL 8.0                | 主存储 |
| 缓存 | Redis 7.x                | 会话、热点数据、分布式锁 |
| 搜索 | Elasticsearch 8.x        | 全文检索 |
| 消息队列 | RabbitMQ 5.x             | 异步解耦、事务消息 |
| 注册/配置 | Nacos 2.x                | 服务发现 + 动态配置 |
| 熔断限流 | Sentinel                 | 流量控制 |
| 文件存储 | 阿里云 OSS                  | 图片/文件 |
| 接口文档 | Knife4j (Swagger)        | 在线调试 |
| 链路追踪 | SkyWalking               | 分布式链路 |
| 容器化 | Docker + Docker Compose  | 部署 |
| CI/CD | Jenkins / GitHub Actions | 持续集成 |
